# 新工程 Modbus 40 寄存器映射

Modbus 保持寄存器起始地址为 0，共 40 个 `WORD`。Python 端读取 `address=0, count=40`。

| 地址 | 内容 |
|---:|---|
| 0 | 场景编号，采集块不覆盖，可由调试表或上位机写入 |
| 1 | 群控调度策略 |
| 2-4 | 1-3号梯当前楼层 |
| 5-7 | 1-3号梯运行方向 |
| 8-10 | 1-3号梯当前载重量 |
| 11-13 | 1-3号梯状态位图 |
| 14 | 上行外呼登记位图 |
| 15 | 下行外呼登记位图 |
| 16-18 | 1-3号梯轿厢内选位图 |
| 19-21 | 分配给1-3号梯的上行外呼位图 |
| 22-24 | 分配给1-3号梯的下行外呼位图 |
| 25-27 | 1-3号梯输出动作位图 |
| 28-30 | 1-3号梯门和安全输入位图 |
| 31 | 群控扫描计数器 |
| 32 | 外呼超时重分配秒数 |
| 33 | 服务中电梯位图 |
| 34 | 故障电梯位图 |
| 35 | 满载电梯位图 |
| 36 | 检修电梯位图 |
| 37 | 系统状态位图 |
| 38 | 建筑最高楼层 |
| 39 | 电梯总数 |

## 位图说明

- 楼层位图：1楼为 bit0，2楼为 bit1，依次到10楼 bit9。
- 地址11-13：bit0服务、bit1故障、bit2满载、bit3检修、bit4电机、bit5上行、bit6下行、bit7高速、bit8低速、bit9开门、bit10关门。
- 地址25-27：bit0电机、bit1上行、bit2下行、bit3高速、bit4低速、bit5开门、bit6关门、bit7-9三级制动、bit10照明、bit11风扇、bit12满载指示、bit13故障指示。
- 地址28-30：bit0开门按钮、bit1关门按钮、bit2光幕、bit3轿门锁、bit4开门限位、bit5关门限位、bit6上平层、bit7下平层、bit8检修、bit9-10上端限位、bit11-12下端限位。
- 地址33-36：bit0、bit1、bit2分别代表1、2、3号梯。
- 地址37：bit0允许节能待机、bit1自动运行、bit2准备就绪。

## 当前缺失

`DB_全部电梯` 使用 `UDT_单梯状态`，但当前导出目录没有该 PLC 数据类型定义，因此第一版没有采集单梯目标楼层、门状态枚举等 UDT 内部字段。后续从 TIA 导出 `UDT_单梯状态` 后，可以替换地址31、32、38、39中的低优先级字段。
