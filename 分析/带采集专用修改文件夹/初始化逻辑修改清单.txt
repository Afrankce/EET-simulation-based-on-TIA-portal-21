初始化逻辑修改清单
====================

一、FB8_双向初始化

1. 在 Static 区保留原变量，并新增：
   平层确认TON        TON_TIME
   端站稳定计数       Int，起始值 0

2. 在 Temp 区保留原变量，并新增：
   上端限位有效       Bool
   下端限位有效       Bool

3. TIA 块编辑器中不要把 FUNCTION_BLOCK、VAR、END_VAR 粘到程序体。
   变量在上方接口表添加，程序体从“段1：公共条件”开始粘贴。

二、FB14_楼层计数

接口和 Static 变量不变，只替换程序体。
新逻辑在端站校准时清空旧平层触发，并把稳定计数封顶为 5，防止 Int 溢出。

三、FB7_单梯控制

删除检修期间每扫描周期调用 FC_检修清理 的程序段。
FB2_检修控制已经负责进入和退出检修时清理输出；重复清理会覆盖检修控制和初始化状态。

四、编译和下载顺序

1. 编译 FB14_楼层计数。
2. 编译 FB8_双向初始化。
3. 编译 FB7_单梯控制。
4. 执行“编译软件（全部重建）”，让三个单梯控制实例 DB 更新多重实例结构。
5. 将程序块和三个单梯控制实例 DB 一起下载到 PLC。

五、监视验证

分别监视三个单梯控制实例 DB 内的：
双向初始化.当前步骤
双向初始化.端站稳定计数
电梯.当前楼层
电梯.上行接触器
电梯.下行接触器
电梯.高速接触器
电梯.低速接触器
电梯.准备就绪

默认向下初始化时，正常步骤应为：
20 寻底 -> 21 底端停车延时 -> 22 上行高速 -> 23 上行低速平层 -> 0

三部梯可以处于不同起始楼层。每部梯必须独立完成上述步骤。
OUT.准备就绪信号只有在三部梯的准备就绪都为 TRUE 时才为 TRUE。
